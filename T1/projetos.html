Projetos

Aqui estão alguns projetos que desenvolvi ao longo do curso:

  • Space Invaders

    Este projeto é uma recriação do clássico jogo Space Invaders, desenvolvido inteiramente em C, utilizando a bibilioteca "Allegro".

    O projeto inclui movimentação dos inimigos, disparos de tiro, sistemas de vidas e de pontuação.

  • Banco Imobiliário

    Este projeto é a implementação do clássico jogo de tabuleiro Banco Imobiliário, desenvolvido na linguagem JAVA, utilizando Swing e AWT.

    O jogo possui sistema de compra e hipotéca de propriedade, pagamento de aluguel, salvar/carregar o jogo, entre outras funcionalidades.

  • Cifras de Beale

    Este projeto implementado na linguagem C consiste na codificação e decoficiação de mensagens usando Cifras de Beale.

    Onde é possível:

    • Codificar uma mensagem qualquer contida em um arquivo ASCII usando um livro cifra.

    • Decodificar uma mensagem usando um arquivo de códigos.

    • Decodificar uma mensagem usando o livro cifra.
